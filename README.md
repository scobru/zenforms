# ZenForms | Decentralized E2EE Forms & Polls

ZenForms is an end-to-end encrypted (E2EE), offline-first, decentralized form builder and live poll application built as part of the **Zen Ecosystem**.

![ZenForms Icon](favicon.svg)

## Features

- 🔒 **End-to-End Encryption (E2EE)**: All form definitions, poll options, and submitted responses are encrypted client-side.
- 📋 **Form Builder**: Easily construct custom forms with text, long text, radio choices, checkboxes, scales, and rating fields.
- 📊 **Quick Polls**: Create single-question polls with instant, real-time live voting and bar/pie chart analytics.
- 📱 **Respondent View**: Clean, mobile-friendly interface for respondents filling out forms or casting poll votes.
- 📈 **Response Analytics & Export**: View aggregated response stats, decrypt responses with master key/passphrase, and export to CSV or JSON.
- 🎨 **Zen Design System**: Built with Vanilla HTML5/CSS3, `Outfit` & `IBM Plex Mono` fonts, Tabler Icons, and automatic Light/Dark mode.
- 🌐 **Zero Server Required**: Runs entirely client-side using `zen` graph sync & storage.

## Architecture

- **Fonts**: `Outfit` (sans-serif) for UI & `IBM Plex Mono` for code/keys.
- **Icons**: `@tabler/icons-webfont`.
- **Sync**: Compatible with `@akaoio/zen` graph sync engine.

## Usage

Simply open `index.html` in any web browser or host via static web server (Vercel, GitHub Pages, Netlify).
